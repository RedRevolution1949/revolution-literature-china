突破网络封锁

    目前国内有关部门机构对一些国外的政治性网站进行封锁，网友使用正常的方法无法浏览这些网站。突破网络封锁的方法很多，比如使用代理服务器，使用突破软件，或者使用在线代理。此处向大家介绍这几种方法：使用软件突破网络封锁，使用在线代理网址突破网络封锁、使用代理地址突破封锁。

一、使用软件突破网络封锁

    向读者介绍五种使用软件突破网络封锁的方法：火凤凰、世界通（GPass）、Tor + Privoxy + SocksCap 组合包、无界浏览、自由之门。

1.1 火凤凰（FirePhoenix）

    火凤凰（FirePhoenix）是一个比较好的突破软件。它可以支持目前常见的各种Internet应用程序，例如浏览器（IE及其它浏览器）、FTP、电子邮件等。只要火凤凰软件在运行，Internet应用程序就可以透明访问网络，无须设置代理地址。

    第一步：下载一个火凤凰软件。下载地址1 | 下载地址2 | 下载地址3

    第二步：安装火凤凰软件。如果下载的是压缩文件，请解压之后再安装。

    第三步：启动火凤凰。选[开始]-->[程序]-->[FirePhoenix]-->[火凤凰(FirePhoenix)]，启动程序。或从桌面双击[火凤凰(FirePhoenix)]快捷方式，启动程序。显示下面的对话框：

    程序将开始初始化网络。如果成功，程序下方的状态条上将显示“状态：初始化成功”。

    第四步：点击[启动安全网络]按钮，启动安全网络。启动过程比较慢，请耐心等候。如果启动成功，程序状态条上将显示[安全网络正在运行]。系统托盘区（操作系统右下角显示时间的区域）将显示一个图标。

    此时可以浏览被封锁的网站了。启动IE浏览器，输入网址，即可访问相应的网站。尝试访问下面三个被封锁的网站：

        人民春秋：http://www.maostudy.org

        中国与世界：http://www.zgysj.com

        中国文革研究网：http://www.wengewang.org

    只要火凤凰安全网络在运行，其它Internet应用程序（例如FTP程序、网际快车等）也可以透明访问网络。

    退出火凤凰的方法：在上点击鼠标右键，弹出一个快捷菜单。点击[退出]菜单项，即可退出火凤凰。

    第二步：启动GPass。双击Gpass.exe，将显示下面的对话框：

    第三步：双击GPass窗口上的Internet Explorer图标，打开一个IE浏览器。在浏览器中输入地址，即可访问被封锁网站。请尝试访问下面的网站：

        人民春秋：http://www.maostudy.org

        中国与世界：http://www.zgysj.com

        中国文革研究网：http://www.wengewang.org

    请注意，只有从GPass窗口上打开的浏览器（或由该窗口新建的浏览器窗口）才能访问被封锁网站。如果想让其它程序（例如：FP程序）访问被封锁网络，请将该程序的快捷方式拖动到GPass窗口上放下，然后从GPass上双击该程序，即可透明访问网络。

    GPass程序窗口如果消失，请双击托盘处的图标，将显示GPass窗口。

    退出GPass的方法：在托盘上处上点击鼠标右键，选[退出]菜单项，即可退出GPass。

1.3 Tor+Privoxy+SocksCap组合包

    Tor+Privoxy+SocksCap组合包是一套开放源代码的突破软件，它没有政治背景，与无界浏览、自由之门不同。

    请注意：Tor+Privoxy+SocksCap组合包不支持win95/98，只支持win2000/XP/Me/2003操作系统。

    第一步：下载一个Tor+Privoxy+SocksCap组合包。如果是压缩文件，请展开它。

    第二步：鼠标双击执行软件包目录下的run.bat文件。将显示两个窗口：一个黑色的命令行窗口，一个SocksCap控制台窗口。

SocksCap控制

    在操作系统右下角（显示系统时间处），将显示一个图标。

    第三步：在SocksCap控制上鼠标双击Internet Explorer图标，启动浏览器。在浏览器地址栏输入网址，现在可以浏览被封锁的网站了。请尝试访问下面的被封锁网站：

        人民春秋：http://www.maostudy.org

        中国与世界：http://www.zgysj.com

        中国文革研究网：http://www.wengewang.org

    请注意，如果此时你使用其它方法打开浏览器（例如，从桌面上打开），并不能突破网络封锁。也就是，必须从SocksCap控制台上打开浏览器，或者从已经突破封锁的浏览器上弹出新页面，或在突破封锁的浏览器上新打开页面，才能自由突破封锁！

    关闭Tor+Privoxy+SocksCap组合包的方法：关闭SocksCap控制台窗口、黑色的命令行窗口，然后在操作系统右下角的图标上点击鼠标右键，再点击Exit Privoxy菜单项。如果关闭Tor+Privoxy+SocksCap组合包之后浏览器无法上网，请将浏览器关闭，再启动浏览器即可。

        人民春秋：http://www.maostudy.org

    如果不想使用无界浏览，在图标上点击鼠标右键，在弹出菜单上选“退出”。软件将显示下面的窗口：

    点击“关闭IE并退出”或“保留IE并退出”，退出无界浏览。如果保留IE退出，您可以继续浏览其它网站。

    如果计算机非正常关机或其它原因导致浏览器无法正常浏览网页（不使用无界浏览时），请按照以下方法恢复浏览器设置：

    1。启动一个浏览器。选“工具/Internet选项”，显示一个弹出窗口－－Internet选项窗口。在窗口上方点击“连接”。

    2。点击“局域网设置(L)...”按钮。弹出一个局域网设置窗口。取消“为LAN使用代理服务器”复选框前的选择标志（√）。

    请注意：无界浏览只能自动识别微软公司的IE浏览器。如果您使用其它的浏览器（如Opera），请启动“无界浏览”软件之后。将浏览器的代理服务器设置为“127.0.0.1”，端口为“9666”。

二、使用在线代理网址突破网络封锁

   第二步：在“URL”后的输入框内输入要到达的目标网址（比如http://www.zgysj.com/)，回车。

 2.3 在网上搜索在线代理

    在线代理网址可能会失效（例如，被封锁），这时就需要寻找新的代理网址。可以使用Baidu、Google等搜索引擎查找新的代理网址。

  • 如果想自己找在线代理，可以搜索“在线代理”或“online web proxy”
  • 如果要英文的在线代理，可以用goole搜“online web proxy”

三、使用代理地址上网

    第一步：找一个代理地址。代理中国网（http://web.proxycn.com/）上有很多代理地址（点击左边的“最新国内http代理”链接或其它类似链接）。例如：60.191.251.9：8080就是一个代理地址－－其中60.191.251.9是代理IP地址，8080是端口。

    第二步：设置代理。启动一个浏览器，选[工具]-->[Internet选项]菜单项，弹出一个对话框。点击[连接]，再点击[局域网]按钮，显示下面的窗口：

    点击选中[为LAN使用代理服务器...]前的复选框，输入地址（如：60.191.251.9），端口（如8080），点击确定按钮，再点击Internet选项窗口的确定按钮。

    第三步：现在可以上网了。在浏览器上输入地址，即可访问被封锁的网站。请尝试访问下面的网站：

        人民春秋：http://www.maostudy.org

        中国与世界：http://www.zgysj.com

    如果不想使用代理地址，请照上述方法，取消代理（将[为LAN使用代理服务器...]前的复选框设置为未选中）。

    代理地址可能会实效，一般可能需要多次尝试才能找到可用的代理地址。
